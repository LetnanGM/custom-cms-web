### Installing

after setup your machine, clone my repo to get the application.

```bash
git clone https://github.com/LetnanGM/custom-cms-web
```

then change directory and install dependencies

```bash
cd custom-cms-web
```

install dependencies

```bash
pip install -r requirements.txt
```

if you error on `pip` command, search it on google :v
it doesnt from my app :D

run the application after installing depedencies!!

```bash
python app.py
```
